a few hot keys for me to remeber.
---------

fn+[ = up

fn+/ = down

fn+; = right

fn+' = left

fn+1 = f1

fn+2 = f2



emacs part
---------

ctrl+f =forwad

ctrl+b = backward

ctrl+p = up

ctrl+n = down

ctrl+d = delete 

ctrl+ u = delete line beginning before cursor

ctrl+k = delete cursor to line end

ctrl+a = jump to the beginning of line

ctrl+e = jump to the end of line.
